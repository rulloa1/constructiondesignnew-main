# Pool Estate Project Images

This folder contains images for the luxury pool estate project.

## Image Naming Convention

To add your pool images **in order**, use this naming pattern:

### Cover Photo
- `pool-cover.jpg` - The main cover image (luxury pool at night with illuminated tree)

### Sequential Images (in order)
- `pool-1.jpg` - Pool area with outdoor TV and stone structure
- `pool-2.jpg` - Curved pool with artificial turf and lounge area
- `pool-3.jpg` - Illuminated tree at night
- `pool-4.jpg` - Pool at night with stone building and covered patio
- `pool-5.jpg` - Pool and dining patio at night with pergola

## Instructions

1. **Save your image files** to your local computer
2. **Rename them sequentially** following the pattern above
3. **Place them in this folder:** `src/assets/projects/pool-estate/`
4. Ensure consistent file format (all `.jpg` or all `.png`)

## Usage in Code

Once images are added, they will be imported in `src/data/projects.ts` and used in the pool estate project entry.

## Image Specifications

- **Recommended format:** JPG or PNG
- **Recommended size:** Optimized for web (typically 1920x1080 or similar)
- **File naming:** Sequential numbering starting from `pool-1.jpg`
- **Cover image:** Always name as `pool-cover.jpg`
